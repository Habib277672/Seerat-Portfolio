🌐 Seerat Portfolio Website

Designed and developed a fully responsive and interactive personal portfolio website to showcase projects, skills, and achievements.

This portfolio is built using HTML, CSS, and JavaScript, with enhanced user experience powered by Anime.js for smooth animations and Swiper.js for dynamic sliders. The project focuses on clean UI design, structured layout architecture, and performance optimization.

Key highlights:

Responsive design for all screen sizes

Smooth and engaging UI animations using Anime.js

Interactive sliders and transitions with Swiper.js

Well-structured semantic HTML and modern CSS styling

Version control and deployment managed using Git & GitHub

Hosted live on Netlify

This project reflects my ability to combine front-end fundamentals with animation libraries to create visually engaging and user-friendly web experiences.

🔗 Live Demo: https://seerat-portfolio.netlify.app/
